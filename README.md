# ToDo

A simple checklist made using HTML, Javascript and CSS.

ToDo can be accessed using this [link](https://todo.admbot.repl.co).

## Updates in V2.1

- Added support for hotkeys.

    `→` to open calender.

    `←` to close calender.
- Updated Visuals.
- Tweaked animations.
